# Cloud Security Portfolio

This portfolio contains hands-on AWS cloud security labs focused on identity and access management, least privilege, cloud infrastructure, logging, monitoring, and secure cloud design.

## Labs

### AWS High Availability Web Architecture Lab
Built a scalable AWS web architecture using Terraform, EC2, an Application Load Balancer, Auto Scaling Group, VPC networking, public subnets, security groups, and multiple Availability Zones.

### AWS IAM Least Privilege Security Lab
Created a least-privilege IAM security lab using a custom IAM policy, IAM user group, test user, and S3 bucket access controls. Configured read-only access to a specific S3 bucket while denying upload, delete, and unrelated AWS service access.

## Skills Demonstrated

- AWS Identity and Access Management
- Least privilege access control
- S3 bucket permissions
- IAM groups and custom policies
- Security validation testing
- Access denied testing
- Cloud security documentation
